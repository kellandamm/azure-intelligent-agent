Private Network Topology: App Service connects to Azure SQL exclusively through VNet integration + private endpoint. No public internet path to SQL. Satisfies Azure Policy requirements out of the box.

Request & Agent Flow (7 Steps)

1

Client Request

Browser or API client sends a chat/REST request over HTTPS. Azure AD Bearer token passed for authenticated routes.

Browser / API ClientAzure AD (Entra ID)
2

FastAPI Ingress, Auth & Rate Limit

Azure App Service (Python 3.11 + Gunicorn) receives the request. auth_service.py validates the JWT; rate_limiter.py enforces per-user throttling to protect OpenAI token budgets.

Azure App Servicemain.pyauth_service.pyrate_limiter.py
3

Session Load from SQL

chat_service.py loads the conversation history for the session from Azure SQL Database via private endpoint. Row-Level Security ensures each user only sees their own data.

chat_service.pyAzure SQL DatabaseVNet / Private Endpoint
4

Agent Orchestration & Secret Resolution

agent_framework_manager.py selects the agent strategy: direct OpenAI, Fabric specialist, or AI Foundry. Secrets fetched from Key Vault via Managed Identity.

agent_framework_manager.pyAzure Key VaultManaged Identityconfig.py
5

AI Inference — GPT-4o Function-Calling Loop

agent_tools.py dispatches tool calls to Azure OpenAI GPT-4o. The agent reasons over conversation history + tool results in a loop until a final answer is produced. Streaming is supported.

agent_tools.pyAzure OpenAI (GPT-4o)
6

Tool Calls: SQL, Fabric, Analytics, Sales

During the agent loop, tools query Azure SQL for structured business data, optionally call Microsoft Fabric specialist agents for lakehouse analytics, and run domain logic via analytics_service.py and routes_sales.py.

Azure SQLMicrosoft Fabric (optional)analytics_service.pyroutes_sales.py
7

Persist, Emit Telemetry & Respond

Final answer is written back to SQL (session history). Telemetry pushed to Application Insights. Streaming/JSON response returned to client. Power BI Embedded reports surfaceable inline.

Azure SQLApplication InsightsPower BI Embedded (optional)

🛡 Network & Security Infrastructure
Azure Virtual Network (VNet)
App Service VNet integration. All outbound traffic to SQL routes through the private subnet. No public SQL exposure.
