# QUICK_START update draft

## Choose a deployment profile

- **Base**: start with SQL-only live analytics.
- **Analytics**: enable Fabric and use Bronze/Silver/Gold plus Direct Lake.
- **Operations**: add RTI for Eventstream, Eventhouse, Activator, and near-real-time dashboards.
- **Governed**: add Purview for catalog, lineage, and governance.
- **Full**: add Foundry + Fabric Data Agent for conversational analytics.

## Default recommendation

Start with Base or Analytics, then add RTI, Purview, or Foundry/Data Agent only when needed.
